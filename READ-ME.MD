    - 1️⃣ What is the difference between var, let, and const?
    Ans: var= Old way to declare a variable. It can be redeclared and updated.
         let = New way to declare a variable. It can be updated, but cannot be redeclared in the same block.
         const = Used for values that should not change. It cannot be updated or redeclared.
